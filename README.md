# Redis 公共工具
```$xslt
        <dependency>
            <groupId>com.github.ddphin</groupId>
            <artifactId>ddphin-redis-spring-boot-starter</artifactId>            
            <version>1.0.0</version>
        </dependency>
```
## 功能
`ddphin-redis`提供公共的redis配置
- redis 缓存
- redis 序列化：使用fastjson
- redis 工具Bean：RedisHelper，随处注入即可使用
